## Sensitive data

Use environment variables to set sensitive data.

### NevaTravel

```
NEVATRAVEL=<API key>
```

### AstraMarin

```
ASTRAMARIN=<Basic Auth token>
```

or

```
ASTRAMARIN_USERNAME=<username> ASTRAMARIN_PASSWORD=<password>
```
